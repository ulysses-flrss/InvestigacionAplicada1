Buenas prácticas de edición de código en JS

Para empezar a dar unos pequeños tips o consejos al momento de codificar o editar en tu editor favorito con javascript daremos una pequeña introducción “¿Que es?” y “¿Que puedes hacer y realizar?”.

Javascript es un lenguaje de programación o que permite implementar funciones o comandos complejos a las páginas web, este lenguaje participa de lo que piensa, está involucrado, en mapas interactivos, en animaciones, actualizaciones, para la interacción, para reproductores de vídeos en los menos te esperas y con las demás capas HTML y CSS pueden complementar un proyecto muy completo y bonito.

Entonces los que puedes realizar es almacenar valores dentro de variables, operaciones sobre fragmentos de textos, ejecutas códigos a respuesta de un click. bueno después de dar esta pequeña introducciones nos damos paso a los consejos.

  • El primer consejo es utilizar "===" en lugar "==" ya que es muy utilizado y utiliza dos tipos diferentes de operadores de igual ya que son muy parecidos para marcar falso en Javascript.

  • El segundo consejo para la codificación de javascript es utilizar Eval en lugar Bad esta función nos da acceso al compilador de javascript, con ellos podemos ejecutar el resultado de una cadena.

  • El tercer consejo es utilizar JSlint, esto es un depurador escrito por Douglas, copias y pegas tu script en este programa y te buscara si tienes algún error en tu código.

  • El cuarto consejo es separarlos de los diferentes capas de CSS y HTML siendo así que en el código solo este javascript ya que esto evita confusiones y es mucho más fácil entenderlo para que quien lo vea.

  • Otro consejo muy importante no utilices variables globales así evitamos y reducimos malas interacciones con otras bibliotecas así mejor declamo la variable dentro de la función.

  • Un consejo muy importante comenta tu código te será más fácil modificar y editar por si lo necesitas, asi si volves a continuar tu proyecto ser mucho más fácil recordarlo.

  • Usa las llaves {} en lugar de New Object() Existen diversas formas de crear objetos en este lenguaje es mas realizarlo así ya que algunos lo llaman mala práctica.

  • Utiliza var con comas, que quiere decir eso, que en lugar de ir una por una declarando las variables, es mejor separarlas con comas es mucho más fácil y sencillo, así limpiamos el código.

  • Este consejo está de más de decilo, pero siempre utiliza siempre pero siempre el punto y coma.

  • Un consejo que es muy importante es que sea autodidacta, que practique ejercicios, vea videos por internet y así desarrollará su lógica para programar.

  • Y para terminar utiliza el editor de texto con el cual te sientes más cómodo, pero nuestra recomendación es Visual Code, Notepad++ y SublimeText.
